# Post Setup Script made by Swayam
### Setup your freshly installed Archlinux System in minutes.

- Single script to rule them all and make it like home.
- No manual hunting of packages and running repeatative commands on multiple systems/containers.
- Easy to modify and make it that truely yours.

-------------------------
<br>

> run this in your terminal as sudo.
```bash
curl -fsSL https://raw.githubusercontent.com/Swayam-SMishra/post-arch/refs/heads/main/post-arch-install-setup.sh | sudo bash
```
<br>

> Need to see what the script runs ? Just curl it and inspect..
> Decided to run ? simply, change the script permission to make it executable...
> And give it root priviledges to install your desired selected packages...
```bash
curl -fsSL https://raw.githubusercontent.com/Swayam-SMishra/post-arch/refs/heads/main/post-arch-install-setup.sh -o post-arch-install-setup.sh
chmod +x ./post-arch-install-setup.sh
sudo ./post-arch-install-setup.sh
```
